Alien Capture

I made this Javascript 3D game with a friend of mine, Ernesto, and the help of a designer for the background graphics. The sounds are open source.

We used threejs to generate and animate the Javascript 3D models.

The code is open sourced on Github.

The game works mostly in Chrome and little else. Feel free to pick an issue and improve it!
