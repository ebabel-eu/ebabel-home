Fun Javascript projects

Alien Capture

I made this Javascript 3D game with a friend of mine, Ernesto, and the help of a designer for the background graphics. The sounds are open source.

We used threejs to generate and animate the Javascript 3D models.

The code is open sourced on Github.

The game works mostly in Chrome and little else. Feel free to pick an issue and improve it!

jsCryptography

I programmed this AngularJS web app to hash any e-mail and find its associated Gravatar (if any).

The code is available on Github.

The web app is also deployed to Github pages by pushing to the gh-pages branch, since the whole web app is client side and doesn't require server side processing.

Give it a try, put in your e-mail to get its MD5 hash and see your Gravatar!
